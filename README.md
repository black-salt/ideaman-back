## IdeaMan 后端项目

### 基于以下组件

* node 
* koa2
* gulp 
* typescript 
* json web token
* socket.io
* pm2
* log4js

### 安装

``` bash
npm install
```

### 运行

``` bash
npm start          # develop
npm run build      # build
npm run eslint     # eslint
npm run server     # run production server
npm run pm2        # pm2 monitor
npm run pm2:dev        # pm2 monitor
```
